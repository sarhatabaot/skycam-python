# skycam

uv workspace monorepo.

Packages:
- skycam-common (shared library)
- skycam-cli (CLI tool)
- skycam-web (future web app)

# skycam-cli

Skycam command-line interface.

## Install (GitHub Releases)

This project is distributed via **GitHub Releases** (no `uv` required for installation).

### Quick install (latest release)

```bash
curl -fsSL https://raw.githubusercontent.com/sarhatabaot/skycam-python/main/scripts/install.sh \
  | REPO="sarhatabaot/skycam-python" TAG="latest" bash
```
